# Overview

This repository documents the complete engineering journey behind our WRO Future Engineers robot, from the earliest brainstorming and conceptual design stages to competition-ready iterations.

It captures the team's growth throughout the project and serves as a record of the countless hours invested in designing, building, testing and refining the robot.

More than a collection of code and hardware, this repository serves as the technical record of our engineering process.

Every major design decision, software revision, mechanical iteration, experiment, failure, milestone and improvement is documented to preserve both the evolution of the project and the engineering reasoning behind it.
