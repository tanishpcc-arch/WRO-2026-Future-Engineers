# Overview## Overview

This repository documents the complete engineering journey behind our WRO Future Engineers robot, from the earliest brainstorming and conceptual design stages to competition ready iterations. It captures the team's growth throughout the project and serves as a record of the countless hours invested in designing, building, testing, and refining the robot.
More than a collection of code and hardware, this repository is intended to be a technical record of our engineering process. Every major design decision, software revision, mechanical iteration, experiment, failure, milestone, and improvement is documented with the aim of creating a clear history of the project's evolution and the reasoning behind our choices.

Our goal is not only to build a competitive robot, but also to develop disciplined engineering practices by documenting the complete development lifecycle in a structured and reproducible manner.
